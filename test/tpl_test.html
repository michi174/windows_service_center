Templatesystem

Diese Daten werden mit einem dynamischen und eigens programmierten Templatesystem ausgegeben. In den folgenden Absätzen werden verschiedene Funktionalitäten des Systems getestet.

Statische Variablen:

Name: {vorname} {nachname}
Alter: {alter} Geschlecht: {geschlecht}

Informationen zur Zeit:

Heute ist {datum} und es ist {zeit}.
Zeitzone: {zeitzone} {zeitunterschied}
Kalenderwoche {kw} Tag {kt}

Browserinformationen:

{browser}

If Test:

Der Besucher ist {if="isset($_SESSION['loggedIn'])"} eingeloggt {else} nicht eingeloggt {/if}.
{if=" '1' == '1' "} 1 ist 1 {/if}

Array Test:

{testarray:vn} {testarray:nn}

Foreach Test:

{foreach testname key=firstname value=lastname} {firstname} {lastname}
{/foreach}
{foreach testnamen_sub key=erster value=letzter} {erster} {letzter}
{/foreach}

Kombination Array, Foreach und If:

{foreach userary key=datakey value=userdata} {userdata:firstname} {userdata:lastname}{if="'{userdata:firstname}' == 'Dominik'"} - der muss sterben *gg*{/if}
{/foreach}

SQL Test:

{datarow sql_test} {sql_test.ID}
{/datarow}
